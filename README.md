# 🧪 Primeiro Projeto QA - Teste Manual

## 📖 Descrição

Este projeto tem como objetivo demonstrar um exemplo prático de teste manual aplicado em um sistema simples.
O foco está na criação, organização e execução de casos de teste documentados em arquivo `.txt`.

---

## 🎯 Objetivo

* Praticar testes manuais
* Documentar casos de teste de forma clara
* Simular atividades reais de um QA
* Criar material para portfólio profissional

---

## 🧪 Tipo de Teste

* ✅ Teste Manual
* ✅ Teste Funcional

---

## 🛠️ Ferramentas Utilizadas

* Editor de texto (.txt)
* Git
* GitHub

---

```
📁 primeiroProjeto
```

## 📋 Exemplo de Caso de Teste

| ID     | Cenário        | Resultado Esperado       |
| ------ | -------------- | ------------------------ |
| CT-001 | Login válido   | Usuário acessa o sistema |
| CT-002 | Senha inválida | Exibir mensagem de erro  |

---

## ▶️ Como Executar o Teste

1. Abrir o arquivo `exemplo de texte manual.txt`
2. Seguir os passos descritos no caso de teste
3. Executar manualmente no sistema
4. Comparar o resultado esperado com o obtido

---

## 🐞 Possíveis Bugs Identificados

* Sistema aceita campos vazios
* Falta de validação de senha
* Mensagens de erro pouco claras

---

## 📊 Resultados Esperados

* Identificação de falhas no sistema
* Validação das regras de login
* Documentação dos testes realizados
